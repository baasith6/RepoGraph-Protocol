# RepoGraph Protocol v0.1.0

## Overview

RepoGraph defines a standard `.repograph/` directory structure that provides machine-readable context about a software repository's architecture, modules, rules, and AI instructions.

## Directory Structure

```txt
.repograph/
  project.yml
  modules.yml
  architecture.yml
  rules.yml
  tests.yml
  ai.yml
  decisions/
    ADR-*.md
  generated/
    graph.json
    graph.mmd
    context-pack.md
```

## Core Files

### project.yml

Defines protocol version and project metadata.

```yaml
protocol:
  name: RepoGraph
  version: 0.1.0

project:
  name: MyProject
  description: Project description
  type: Modular Monolith
  architecture: Clean Architecture
  primary_language: C#
```

### modules.yml

Maps business modules to file path globs.

```yaml
modules:
  - name: Auth
    description: Authentication module
    paths:
      - "src/**/Auth/**"
    critical: true
```

### architecture.yml

Defines architecture layers and allowed dependencies.

```yaml
architecture:
  style: Clean Architecture

layers:
  Domain:
    path: "src/Domain"
    allowed_dependencies: []

  Application:
    path: "src/Application"
    allowed_dependencies:
      - Domain

rules:
  - id: no-web-to-infrastructure
    description: Web must not depend on Infrastructure
    severity: error
```

### rules.yml

Custom architecture rules and enforcement mode.

```yaml
rules:
  - id: custom-rule
    description: Rule description
    severity: error
    type: forbidden-dependency
    from: Web
    to: Infrastructure

enforcement:
  mode: error  # warning | error | strict
```

### ai.yml

Instructions for AI coding tools.

```yaml
ai:
  instructions:
    - Follow architecture layer rules
  exclude_paths:
    - "node_modules/**"
  risk_warnings:
    - Auth module is critical
```

## Generated Output

### graph.json

Standard JSON graph containing nodes, edges, violations, and stats. See `packages/graph-core/src/types.ts` for the schema.

### graph.mmd

Mermaid diagram of module and layer relationships.

### context-pack.md

Markdown context pack for AI tools, including project summary, modules, rules, and violations.

## JSON Schema

Machine-readable schemas are in `packages/protocol/src/schemas/`.
