# v0.5.0 — .NET depth and distribution

## Install

```bash
npm install -g @repographprotocol/cli@0.5.0
```

Requires **Node.js 20+** and **.NET SDK** for Roslyn-based C# / EF analysis.

## Added

- **EF Core mapping** — Roslyn extracts `DbSet`, entities, migrations → enriched `database.yml` (`table`, `tenant_scoped`, `required_fields`, `migration_files`)
- **Bundled Roslyn tool** — `roslyn-tool/` shipped in npm CLI bundle for global installs
- **`@repograph/analyzer-angular`** — Angular route extraction (compiler API + regex fallback)
- **CI templates** — [Azure DevOps](../ci/azure-pipelines-repograph.yml), [GitLab](../ci/gitlab-repograph.yml)
- **Integration docs** — [MCP](../integrations/mcp.md), [VS Code](../integrations/vscode.md)
- **MCP package** — `@repograph/mcp-server` prepared for npm publish (no longer `private`)

## Changed

- `repograph scan` syncs Angular routes into `api.yml` when detected
- CLI version **0.5.0**
