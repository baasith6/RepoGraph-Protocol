# v0.1.0 — Initial release

First implementation of the RepoGraph Protocol monorepo.

## Added

- Protocol v0.1 specification and JSON schemas
- CLI: `init`, `validate`, `doctor`, `scan`, `sync`, `check`, `explain`, `list`, `visualize`, `stats`, `export`, `prompt`, `impact`
- Heuristic C# and TypeScript analyzers
- Graph builder, rule engine, exporters (json, markdown, mermaid, cursor)
- Example: Clean Architecture C# + Angular
