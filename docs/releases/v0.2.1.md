# v0.2.1

## Install

```bash
npm install -g @repographprotocol/cli@0.2.1
```

**Important:** `@repographprotocol/cli@0.2.0` fails at runtime on global install. Use **0.2.1** or later.

## Fixed

- npm-published CLI runtime crash by bundling to CommonJS output (`dist/index.cjs`) with an ESM shim entrypoint (`dist/index.js`).
