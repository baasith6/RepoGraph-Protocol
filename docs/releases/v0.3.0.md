# v0.3.0

## Install

```bash
npm install -g @repographprotocol/cli@0.3.0
```

## Added

- **Incremental scan cache** — `scan-cache.json` in `.repograph/generated/` reuses unchanged files on `repograph scan`
- **Graph drift detection** — `repograph validate` warns when `graph.json` is stale vs the repo (`--strict` to fail)
- **scanSignature** stored in `graph.json` stats for drift checks
- Demo GIF on README (`docs/assets/demo.gif`)

## Changed

- CLI version **0.3.0**
- GitHub Action pins `@repographprotocol/cli@0.3.0`
