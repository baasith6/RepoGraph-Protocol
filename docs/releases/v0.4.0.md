# v0.4.0 — AI Context

## Install

```bash
npm install -g @repographprotocol/cli@0.4.0
```

## Added

- **`@repograph/context-engine`** — task-aware module/file/rule/risk selection
- **`repograph prompt --mode short|full|strict`** — structured task context (not full repo dump)
- **`repograph impact --json`** — rich blast radius with APIs, tests, reviewers
- **`repograph export --format claude|copilot`** — `CLAUDE.md` and `.github/copilot-instructions.md`
- **`context-pack.short.md`** on scan — compact AI context artifact
- **MCP** `repograph_prompt` tool
- **VS Code** command: Copy Task Prompt

## Changed

- `repograph impact` uses reverse `DEPENDS_ON` traversal and `api.yml` / `ownership.yml`
- CLI version **0.4.0**
